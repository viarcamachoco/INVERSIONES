# Constitución del Proyecto
## DIANACORE‑CONSTITUTION

Propósito:
Definir los principios, límites y reglas fundamentales del proyecto DIANA‑SDK‑Core,
encargado de proveer herramientas, métricas y gobernanza
para Spec‑Driven Development asistido por IA.

Principios rectores:
- El canon gobierna al código
- No se permite inferencia automática de requisitos
- Speckit ejecuta, no decide
- La calidad SDD debe validarse antes de la ejecución

Alcance del proyecto:
- Métricas de calidad SDD
- Checklists automáticos
- Dashboards de calidad
- Auditoría de iniciativas y ejecución

Fuera de alcance:
- Gestión de tickets como sistema
- Lógica de negocio de productos finales
- Desarrollo sin especificación explícita

IA:
La IA actúa como copiloto.
Toda acción debe ser trazable, auditable y reversible.

Idioma:
Español como idioma principal.

Estado:
Constitución canónica del proyecto DIANA‑SDK‑Core.
