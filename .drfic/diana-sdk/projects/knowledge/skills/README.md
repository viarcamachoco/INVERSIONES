# Cross-Project Skills — Projects Root

Este directorio puede alojar skills reutilizables entre proyectos del mismo ecosistema de negocio.

Convencion recomendada:
- `<NNN>-<dominio>-<skill-name-predictivo-en-minusculas>.md`

Para el proyecto de inversiones usar:
- `<NNN>-inv-<skill-name-predictivo-en-minusculas>.md`

Regla de resolucion:
1. Cargar skills de `projects/knowledge` (si aplica).
2. Cargar skills del proyecto activo.
3. El proyecto activo tiene prioridad final.
