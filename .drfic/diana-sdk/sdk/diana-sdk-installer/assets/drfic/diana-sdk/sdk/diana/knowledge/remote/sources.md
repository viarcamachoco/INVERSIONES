# Fuentes Remotas — Diana SDK Knowledge (SDK Level)

> Catálogo de fuentes externas sobre SDD, metodología de agentes, y diseño de software con IA.

## Cómo usar este catálogo

1. Para capturar un snapshot de una fuente: `/diana.knowledge topic="[tema]" scope="sdk" type="remote"`
2. Los snapshots se guardan en `../snapshots/`
3. Formato de snapshot: markdown exportado o resumen estructurado
